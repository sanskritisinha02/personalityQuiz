# personalityQuiz
Guided Project : Personality Quiz 


In this guided project, we created a personality quiz. It is a game where players are presented with a lighthearted topic and answer questions that align them to a particular outcome. There are no correct answers to quiz questions. Answers are purely subjective, and their results donâ€™t even have to be logical.
